# 🛡️ PromptShield ML

## Multi-Class Prompt Injection Detection using NLP & Machine Learning

PromptShield ML is an AI Security project that detects and classifies prompt injection attacks using traditional Natural Language Processing (NLP) and Machine Learning techniques.

The project was developed to demonstrate how prompt injection attempts can be automatically identified before reaching a Large Language Model (LLM), making it suitable as a lightweight security layer for AI applications.

---

## 🚀 Project Objective

The objective of PromptShield ML is to build an end-to-end Machine Learning pipeline capable of identifying different categories of prompt injection attacks while distinguishing them from legitimate user prompts.

Instead of focusing on LLM response generation, this project focuses on the security of user inputs before they are processed by an LLM.

---

## ✨ Features

- Custom Prompt Injection Dataset
- Exploratory Data Analysis (EDA)
- Prompt Length Analysis
- Class Distribution Visualization
- TF-IDF Feature Engineering
- Linear Support Vector Machine (Linear SVM)
- Multi-Class Prompt Classification
- Confusion Matrix
- Classification Report
- Model Serialization using Joblib
- Interactive Prompt Prediction

---

## 📂 Dataset

A custom synthetic benchmark dataset was created specifically for this project.

Dataset Characteristics:

- 1000 Prompt Samples
- Balanced Prompt Categories
- Safe Prompts
- Prompt Injection Prompts
- AI Security Focused

The dataset was intentionally designed as a synthetic benchmark to simulate common prompt injection patterns for educational and research purposes.
